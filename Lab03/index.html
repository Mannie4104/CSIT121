What I learned

Lab 03

Published on February 23, 2024 by Emmanuel

Styling my first webpage was an exciting experience. I was thrilled to have the opportunity to bring my ideas to life and make the design visually appealing. With the help of CSS (Cascading Style Sheets), I was able to customize the background color and font size. It felt nice to see my webpage transform from a simple structure into something a little more engaging.

What I learned

  1. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  2. The three main parts of a CSS rule are HTML selectors, ID selectors, and class selectors.
  3. An ID selector should be used when you want to apply a style to just one specific element.
